
SylFm プラグイン
================================================================================

このプラグインはSylFilterの補助ツールです。

Sylpheedの迷惑メール対策設定でsylfilter使うことができますが、
ある程度学習するまでは誤判定されることがあります。

そこでコマンドラインからsylfilterを実行せずとも
メールを選択した状態でメニューから
再学習を行えるような機能をSylFmでは提供します。

動作環境
--------------------------------------------------------------------------------

Windows XP SP3
Sylpheed 3.1.2以降
SQLite3

インストール方法
--------------------------------------------------------------------------------

sylfm.dllをSylpheedのプラグインディレクトリへとコピーします。

1) Sylpheedのzipアーカイブを使用している場合

pluginsディレクトリへ配置します。

2) Sylpheedをインスーラによりセットアップした場合

%APPDATA%/Sylpheed/plugins以下に配置します。
Windows XPの場合以下の場所にdllをコピーして下さい。
C:\Documents and Settings\(ユーザアカウント)\Application Data\Sylpheed\plugins

3) SylFilterに同梱されているlibsqlite3-0.dllをSylpheed.exeと同階層にコピーします。
   http://sylpheed.sraoss.jp/sylfilter/index.html.ja

1)もしくは2)の後にlibsqlite3-0.dllを配置すればセットアップは完了です。

使い方
--------------------------------------------------------------------------------

サマリビューもしくはテキストビューからフィルタ結果を表示させることができます。

サマリビューの場合

1) サマリビューで右クリック
2)「sylfilterの状態を表示」をクリック

テキストビューの場合

1) テキストビューで右クリック
2)「sylfilterの状態を表示」をクリック

フィルタ結果画面では手作業でメールに対してspam/通常のメールとして再学習
させることができます。spam/通常メールいずれかを選択してから
適用ボタンをクリックしてください。
